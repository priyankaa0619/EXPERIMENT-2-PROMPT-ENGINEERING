# EXP-2: PROMPT ENGINEERING

## Aim

Comparative Analysis of Different Types of Prompting Patterns and Explanation with Various Test Scenarios.

---

## Experiment

### Scenario 1: Educational Topic

**Broad Prompt:** Explain Artificial Intelligence.

**Response:** AI is a technology that enables machines to perform tasks that normally require human intelligence.

**Detailed Prompt:** Explain Artificial Intelligence for a first-year engineering student with examples.

**Response:** Artificial Intelligence (AI) is a branch of computer science that allows machines to learn, reason, and make decisions. Examples include voice assistants, self-driving cars, and recommendation systems.

**Observation:** Detailed prompt gives a clearer and more informative response.

---

### Scenario 2: Programming Task

**Broad Prompt:** Write an addition program.

**Response:** Provides a simple addition program.

**Refined Prompt:** Write a Python program to add two numbers entered by the user and display the result.

**Response:**

```python
a = int(input("Enter first number: "))
b = int(input("Enter second number: "))
print("Sum =", a + b)
```

**Observation:** Refined prompt produces a complete and usable solution.

---

### Scenario 3: Technical Explanation

**Broad Prompt:** Explain RF testing.

**Response:** RF testing is used to evaluate radio frequency systems.

**Role-Based Prompt:** Act as an RF engineer and explain RF testing methods and applications.

**Response:** RF testing verifies the performance of RF systems using power measurements, spectrum analysis, network analysis, and compliance testing to ensure reliable operation.

**Observation:** Role-based prompting provides more technical depth and accuracy.

---

### Scenario 4: Creative Writing

**Broad Prompt:** Write a story.

**Response:** Generates a short general story.

**Detailed Prompt:** Write a short story about a student who develops a robot for farmers.

**Response:** Generates a focused story with proper plot, characters, and conclusion.

**Observation:** Detailed prompts improve creativity and relevance.

---

## Output

Different prompting patterns generated different levels of response quality. Broad prompts produced general answers, while detailed, refined, and role-based prompts produced more accurate, relevant, and comprehensive responses.

---

## Result

The experiment was successfully conducted. It was observed that structured prompting techniques such as detailed prompts, refined prompts, and role-based prompts provide better accuracy, clarity, relevance, and depth compared to broad or unstructured prompts.
